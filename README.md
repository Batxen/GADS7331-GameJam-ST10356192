# GADS7331-GameJam-ST10356192
This is the GitHub repository for student ST10356192 to be used for the storage and distribution of the finalised Game Jam project build

Overview:
For this GADS7331 module, we are expected from the 25th -> 27th March to develop a playable build of a game as a part of a Game Jam. 
During this time we must ideate, develop, and finalise a working build of a game through the use of AI tools to learn a greater understanding of their strength within day-to-day life as game developers as well as to reflect upon the pros and cons of using AI the way we do.
We have been given the theme of 'Switch' to make our game with. With this theme I am developing a prototype game that sees the player swap control between two different objects: The Player Character used to traverse the environmet, and a Drone Character used to solve puzzles within those same environments.
