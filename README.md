# GADS7331-GameJam-ST10356192
This is the GitHub repository for student ST10356192 to be used for the storage and distribution of the finalised Game Jam project build

Overview:
1.) For this GADS7331 module, we are expected from the 25th -> 27th March to develop a playable build of a game as a part of a Game Jam. 
2.) During this time we must ideate, develop, and finalise a working build of a game through the use of AI tools to learn a greater understanding of their strength within day-to-day life as game developers as well as to reflect upon the pros and cons of using AI the way we do.
3.) We have been given the theme of 'Switch' to make our game with. With this theme I am developing a prototype game that sees the player swap control between two different objects: The Player Character used to traverse the environmet, and a Drone Character used to solve puzzles within those same environments.
4.) Download the contents within the Git repository, within the folder you download will be  executable called GADS7331-GameJam. Double click on the Application executable to begin the game.

Controls:
W - Forward
A - Strafe Left
S - Strafe Right
D - Backwards
Q - Switch to Drone / Switch to Player

While as Player: 
Spacebar: Jump

While as Done:
Spacebar: Ascend
Left Control: Descend
